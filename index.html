Nformd

A single-page web app, inspired by Medium

Nformd is truly a single-page app. I served as the full-stack developer, and used Ruby on Rails with a PostgreSQL database to govern the backend. On the frontend, I used JavaScript with React.js and Flux to maintain a unidirectional flow of data and provide instantaneous feedback for user interactions.

Custom-built front and backend authentication salt and hash user passwords for secure storage and confirmation when users log in.

The Draft.js API allows for a rich text editor configured to parse content as raw JavaScript for persistent storage

Amazon Web Services S3 allows for efficient uploads of custom user profile pictures

Pac-Man

The arcade classic lives again on your desktop

Built using only object-oriented JavaScript and HTML5 Canvas, it's Pac-Man! With retro audio, three-letter high-scores, and simple directional controls, Pac-Man has never looked so...exactly the same.

Separate JavaScript classes abstract characters into public APIs, and the maze into the interactive environment where they live.

Dynamic HTML5 Canvas drawings create each character only once, in each state of their animations.

Cookies save your top 10 high scores locally, within your browser.
